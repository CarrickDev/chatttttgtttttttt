Affichage d'une image

Assurez-vous de remplacer "chemin/vers/votre/image.jpg" par le chemin relatif ou absolu vers votre image. Par exemple, si votre image se trouve dans le même répertoire que le fichier "index.html", vous pouvez simplement spécifier le nom de l'image ("image.jpg"). Si l'image se trouve dans un sous-répertoire, vous devez spécifier le chemin relatif par rapport à "index.html" ("sous-repertoire/image.jpg"). Enregistrez ce code dans un fichier nommé "index.html" et assurez-vous que l'image spécifiée existe à l'emplacement indiqué. Ensuite, ouvrez le fichier "index.html" dans un navigateur web pour afficher l'image.
